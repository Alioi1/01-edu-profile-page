# GraphQL Profile Visualizer

A lightweight single-page application (SPA) built for the **Yessenov University** project. This tool allows students to log in, view their academic progress, and visualize their performance through custom SVG graphs.

## 🚀 Features
- **Secure Authentication:** JWT-based login using Basic Auth (Base64).
- **GraphQL Integration:** Real-time data fetching from the school's GraphQL API.
- **Data Visualization:** Custom-built SVG charts (no external libraries used):
  - **XP Progress:** A graph showing experience points earned over time.
  - **Audit Ratio:** A visual representation of successful vs. unsuccessful audits.
- **Minimalist UI:** Clean, professional design focused on readability.

## 🛠 Tech Stack
- **HTML5** & **CSS3** (Custom styling)
- **Vanilla JavaScript** (ES6+)
- **GraphQL** (API queries)
- **SVG** (Scalable Vector Graphics for data visualization)

## 📁 Project Structure
- `index.html` - The main structure of the application.
- `style.css` - UI components and responsive layout.
- `app.js` - Core logic: authentication, API calls, and SVG generation.
